HTML

HyperText Markup Language

A standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages. (Merriam Webter's Online Dictionary)

How is HTML used?

HTML is used to format text on a webpage, as well as format graphics on a webpage.

Where did HTML come from?

HTML was born with the World Wide Web, created by Tim Berners-Lee at CERN (European Laboratory for Particle Physics) in Geneva, Switzerland. The World Wide Web (WWW) was originally created so that the scientists in this huge hub of science could share their work with others in the work place. The WWW exploded with popularity throughout the 1900s.

Hypertext, the basis of HTML, stemmed from the WWW. In the late 1980s, Bill Atkinson, a programmer for Apple Computer Inc. created the hypercard, based off of hypertext. By using this, applications and softwares were able to take form on screen, controlled and dictated through scripts.

Later on in the 1990s, the internet, formally called the WWW, gained popularity, and many types of browsers, including Montulli's Lynx browser, as well as Sun Microsystem Inc.'s The Mosaic browser were released.

How is HTML written?

CSS

Cascading Style Sheets

defines how HTML elements are to be displays, using HTML.

How is CSS used?

CSS is used in partnership with HTML to format and add aesthetic qualities to a webpage. From coloring, format, shape, and design, the possibilities of what you can create with CSS are endless.

Where does CSS come from?

In short, CSS was derived from programers and creators who were apart of the initial browser/coding movement in the 1990s, including people like Tim Burners-Lee.
